# **App Name**: WeatherGo Navigator

## Core Features:

- User Input: Collect user's gender and personality preferences via single-choice buttons.
- Weather Data Fetch: Fetch weather forecast data for March 13-17, 2026, using a weather API.
- Personalized Recommendation Engine: Recommend travel destinations based on weather conditions (good/bad), gender, and personality, incorporating a 'foolproof mechanism' for bad weather.
- Google Image Search: The 'tool' searches for images of recommended locations using the Google Custom Search API or a mock interface.
- Visual Output: Display weather overview, recommended activities images, and short comments in a card-style format.

## Style Guidelines:

- Primary color: Sky Blue (#87CEEB), evoking a sense of calm and exploration related to travel and weather.
- Background color: Very light cyan (#E0FFFF), maintaining a bright and airy feel, like the sky.
- Accent color: Soft Lavender (#D8BFD8), an analogous hue creating a gentle contrast while complementing the primary color's calming effect.
- Font pairing: 'Poppins' (sans-serif) for headlines, providing a modern, clean, and precise feel, combined with 'PT Sans' (sans-serif) for body text, for readability and warmth.
- Minimalist homepage design with a central label selector and card-based design for the result page, including weather overview, images, and action suggestions.
- Use weather-themed icons, and icons that correspond to the tags, keeping with the playful tone of the app.
- Subtle transition animations for loading recommendations.
